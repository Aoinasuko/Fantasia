# Fantasia(ファンタジア)

## 概要
ファンタジアはローカルLLMを使ってRPGのようなものを行うフリーゲームです。  
Instantaleというゲームから着想を得て制作を行っています。  
このプロジェクトは、Codex(ChatGPT)を使用して開発を行っています。

現在まだWIPの段階のため、まともにプレイできない可能性があります。  
ご了承ください。

## プレイ方法
1:Releaseからプレイ環境をダウンロードします。  
2:model/graphicに好みのSDXLのモデルを、model/textに好みの文章生成のGGUFモデルを入れてください。  
3:Fantasia.exeを起動してください。  

## ビルド方法
scripts/build_exe.ps1を実行すると配布用ビルドを行います。
